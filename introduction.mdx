## Overview

CALZADO J&R is a full-stack management system designed specifically for shoe manufacturing operations. Built with modern web technologies, it streamlines the entire production workflow from customer orders to employee task management and inventory control.

### Three-tier role system

The platform implements a secure, role-based access control system with three distinct user types:

- **Administrators** - Full system access to validate customers, create employee accounts, and manage all operations
- **Employees** - Task-focused dashboard for production workers (Guarnición, Solador, Cortador, Emplantillador)
- **Customers** - Self-service registration with order tracking and catalog access

### Built for production environments

The system is production-ready with Docker Compose orchestration, PostgreSQL database persistence, JWT authentication, and email integration for password management and employee onboarding.
